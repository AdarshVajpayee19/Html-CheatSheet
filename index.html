This HTML CheatSheet Is For You!
  1. HTML Boilerplate : this is the basic template or barehouse structure of HTML. 
    <!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>

</body>
</html>
  2. Headings : There are six Headings Tags available H1 is largest among all,H6 is smallest. 
    <h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
  3. Container : Container Tag are tags that contain some data such as text,images,etc.There are several container tags in html.
      4. (i.)Div Tag :
      div tag or division is used to make blocks or divisions in the document.
      <div>This is Div Block </div>
      (ii.)Span Tag :
      Span is a container for inline content.
      <span>This is span Block </span>
      (iii.)Paragraph Tag : 
      <p>This is Paragraph </p>
      (iv.)Pre Tag : pre tag represents pre-formatted text. 
      <pre>This is Pre text </pre>
      (v.)Code Tag : code tag is used to represent Source code. 
      <pre>import pandas as pd </pre>
  4. Text Formatting :
    Text formatting are used to format text or data of HTML documents.You can do Certain things like creating italic,bold,strong Text to make document look more attractive and understandable.
      5. (i.)<strong> Tag : 
      <strong>I'm Important text. </strong>
      (ii.)<b> Tag : 
      <b>I am Bold Text. </b>
      (iii.)<i> Tag : 
      <i>I'm italics text. </i>
      (iv.)<em> Tag : 
      <em>This is Emphasized text </em>
      (v.)<sub> Tag : 
      <sub> Subscript </sub>
      (vi.)<sup> Tag : 
      <sup>SuperScript</sup>
  5. Lists :
    Lists can be either numerical,alphabetic,bullet, or other symbols.You can specify list type and list items in HTML for the Clean document.
      6. (i.)<ol> Tag : Ordered list starts with <ol> and each list item starts with <li> 
      <ol>
<li>Data 1</li>
<li>Data 2</li>
<li>Data 3</li>
</ol>
      (ii.)<ul> Tag : 
      <ul>
<li>Your Data</li>
<li>Your Data</li>
</ul>
  6. Media :
    Media is anything that is present in digital form such as images,videos,audios etc.
      7. (i.)<audio> Tag :
      It is used to embed Sound content in the Document.
      <audio controls>
<source src="demo.mp3" type="audio/mpeg">
Your browser doesnot support the audio element.
</audio>
      (ii.)<img> Tag : It is used to embed or import image in a webpage 
      <img src="Source_of_image" alt="Alternate_image">
      (iii.)<video> Tag :
      It is used to embed video content in the Document.
      <video width="480" height="320" controls>
<source src="demo_move.mp4" type="video.mp4">
Your browser doesnot support the video element.
</video>

  8. Links :
    Links are clickable text that can redirect you to some other page.
    <a> Tag: Defines an hyperlink 
    <a href="https://www.google.com">Visit Google</a>

  10. Semantic Elements : Semantic Elements are those that are self describable,i.e. from their name itself, you can understand their meaning.
      11. (i.)<section> Tag :
      Its defines a section in the document.
      <section>This is Section </section>
      (ii.)<article> Tag :
      It represents self contained content.
      <article>Enter you data </article>
      (iii.)<aside> Tag :
      It is used to place the content in the sidebar.
      <aside> </aside>
